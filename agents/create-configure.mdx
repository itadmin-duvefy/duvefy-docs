## Create a new agent

<Steps>
  <Step title="Navigate to Agents">
    In the left sidebar, click **Agents**.
  </Step>
  <Step title="Click New Agent">
    Click the **New Agent** button in the top-right corner.
  </Step>
  <Step title="Fill in basic info">
    Complete the required fields (see below) and click **Save**.
  </Step>
</Steps>

---

## Basic information

### Name
The internal name used to identify this agent in your dashboard. Visitors don't see this name — it's for your team's reference.

### Mission
A short description of the agent's purpose, written in natural language. The AI uses this to understand its role and set the right tone.

**Example:**
> "Help visitors learn about our scholarship programs, answer questions about eligibility, and collect donation pledges."

The clearer and more specific the mission, the better the agent performs.

### Agent type
Select **Donation** or **Sales** based on your primary use case. This shapes how the AI approaches conversations.

### Active toggle
New agents start **inactive**. Turn this on when you're ready for the agent to go live.

---

## Tone & rules

You can optionally add:
- **Tone instructions** — How the agent should communicate (e.g., "Always be warm, empathetic, and encouraging")
- **Rules** — Constraints on agent behavior (e.g., "Never discuss competitor organizations", "Always suggest speaking to a human if the user expresses frustration")

These are appended to the agent's system prompt on every conversation.

---

## Footer links

Footer links are optional URL pairs shown at the bottom of the chat widget. Useful for directing visitors to important pages.

| Label | URL |
|-------|-----|
| Donate Now | https://example.org/donate |
| About Us | https://example.org/about |

You can add, remove, and reorder links. They appear in the order you set them.

---

## Saving changes

Click **Save Changes** at the bottom of the page. Changes to the agent configuration (name, mission, tone, rules, footer links) take effect immediately on your live widget.

<Note>
  Widget styling changes (colors, icon, position) are saved separately in the **Styling** section and update the embed script immediately.
</Note>
