## What is an agent?

An agent is a fully configured AI assistant that lives inside a chat widget on your website. Visitors can type questions, and the agent responds using its knowledge base. When a conversation matches a defined workflow (topic), the agent can take actions — like asking follow-up questions, creating a CRM record, or initiating a payment.

---

## Agent types

Duvefy currently supports two agent types:

<CardGroup cols={2}>
  <Card title="Donation Agent" icon="heart">
    Designed for nonprofits and charities. Engages visitors, answers questions about your mission, and processes one-time or recurring donations via Stripe.
  </Card>
  <Card title="Sales Agent" icon="briefcase">
    Designed for sales-led teams. Qualifies leads, collects contact information, and creates CRM opportunities.
  </Card>
</CardGroup>

The agent type affects how the AI frames its behavior and what it prioritizes in conversations.

## Multiple agents

You can create multiple agents within one organization (subject to your plan limit). Each agent can have a completely different knowledge base, style, and set of workflows. Use cases:
- Different agents per department (Fundraising, Events, General Enquiries)
- Different agents per product or landing page
- A/B testing different agent personalities

---

## Active vs. inactive

Each agent has an **Active** toggle. When inactive:
- The widget still loads on your website
- Visitors see a "chat unavailable" state
- No messages are processed or counted against your quota

This is useful for seasonal campaigns or testing.
