## Overview

Every agent has its own widget style. You can choose from preset themes or build a fully custom look using color pickers.

Styling settings are found in the **Styling** tab on the agent edit page.

---

## Preset themes

Six themes are available out of the box:

| Theme | Description |
|-------|-------------|
| **Dark** | Dark backgrounds with light text — sleek and modern |
| **Light** | Clean white backgrounds — minimal and professional |
| **Ocean** | Blue tones — calm and trustworthy |
| **Warm** | Amber and orange accents — friendly and energetic |
| **Purple** | Purple palette — creative and distinctive |
| **Green** | Green tones — fresh and positive |

Select a preset to instantly apply it. You can then fine-tune individual colors using the custom controls.

---

## Custom colors

When you select **Custom**, individual color pickers become available for:

| Element | Controls |
|---------|---------|
| Header background | Chat window top bar |
| Message bubbles | Visitor and agent message colors |
| Send button | The send/submit button |
| Input background | The text input field |
| Font family | Choose from system fonts or Google Fonts |
| Border radius | Rounded corners on the chat window |

## Position

Set where the launcher button appears on the page:

| Option | Description |
|--------|-------------|
| **Bottom right** (default) | Most common; follows standard UX patterns |
| **Bottom left** | Use when your site has a sticky element on the right |
| **Top right** | Less common; useful for header-based layouts |
| **Top left** | Rare; use only for specific design requirements |

---

## Live preview

As you make changes, a scaled preview of the widget updates in real time — including the launcher position and icon. No need to save to see how it looks.

---

## Saving and publishing

Click **Save Changes** to publish your styling updates. The embed script updates immediately — visitors who reload the page will see the new style without requiring a new script deployment.
