## Before you begin

You'll need:
- An active Duvefy account (check your inbox for the invitation email)
- Access to the website where you want to embed the agent

---

## Step 1: Sign in

Go to [app.duvefy.com/login](https://app.duvefy.com/login) and sign in with your email and password.

If you haven't set your password yet, click the link in your invitation email — it will take you to a page where you can create one.

  The Duvefy login page showing the email and password fields. Highlight the "Sign In" button.
</Info>

## Step 2: Create your first agent

<Steps>
  <Step title="Go to Agents">
    In the left sidebar, click **Agents**, then click **New Agent**.

    <Info>
      📸 **Screenshot needed** — `images/quickstart-agents-list.png`

      The Agents page showing the empty state (or existing agents list) with the **New Agent** button highlighted in the top-right corner.
    </Info>
  </Step>
  <Step title="Fill in the basics">
    Enter a **Name** for your agent (e.g., "Website Assistant") and a **Mission** — a short description of what this agent should do (e.g., "Help visitors learn about our services and capture their contact details").

    <Info>
      📸 **Screenshot needed** — `images/quickstart-agent-form.png`

      The new agent form with the Name field, Mission textarea, and agent type selector visible. Highlight the Name and Mission fields.
    </Info>
  </Step>
  <Step title="Add knowledge">
    Under the **Knowledge Base** tab, paste your website URL or some text about your organization. The agent will use this to answer visitor questions.

    <Info>
      📸 **Screenshot needed** — `images/quickstart-agent-kb.png`

      The Knowledge Base tab with the URL input field and the free text area visible. Show a sample URL entered in the URLs field.
    </Info>
  </Step>
  <Step title="Save">
    Click **Save Changes**. Your agent is now configured.

    <Info>
      📸 **Screenshot needed** — `images/quickstart-agent-saved.png`

      The agent edit page after saving, showing the success confirmation and the agent name in the page header.
    </Info>
  </Step>
</Steps>

---

## Step 3: Copy the embed script

<Steps>
  <Step title="Open the Embed tab">
    Inside the agent edit page, click the **Embed** tab.

    <Info>
      📸 **Screenshot needed** — `images/quickstart-embed-tab.png`

      The agent edit page with the tab bar visible, highlighting the **Embed** tab selected.
    </Info>
  </Step>
  <Step title="Copy the script">
    Click the **Copy** button to copy the `<script>` snippet.

    <Info>
      📸 **Screenshot needed** — `images/quickstart-embed-copy.png`

      The Embed tab showing the full script snippet and the **Copy** button highlighted.
    </Info>
  </Step>
</Steps>

The snippet looks like this:

```html
<script
  src="https://app.duvefy.com/widget/YOUR_ORG_ID/embed.js"
  async
></script>
```

---

## Step 4: Add the script to your website

Paste the snippet before the closing `</body>` tag on any page where you want the agent to appear.

```html
<body>
  <!-- your page content -->

  <script
    src="https://app.duvefy.com/widget/YOUR_ORG_ID/embed.js"
    async
  ></script>
</body>
```

<Tip>
  You can also embed the agent as an `<iframe>` for more control over placement and sizing. See [Embed the Widget](/agents/embed) for iframe instructions.
</Tip>

---

## Step 5: Test it

Reload your website. You should see the Duvefy chat launcher in the bottom-right corner. Click it and send a message to confirm the agent responds correctly.
